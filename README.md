This project presents an AI-powered Predictive Maintenance System developed using IBM watsonx.ai AutoAI and IBM Cloud to predict industrial machine failures before they occur. The solution analyzes machine sensor data, including air temperature, process temperature, rotational speed, torque, tool wear, and machine type, to classify different types of equipment failures.

The machine learning model automatically performs data preprocessing, feature engineering, algorithm selection, and hyperparameter optimization using IBM watsonx.ai AutoAI. The best-performing Random Forest Classifier was selected, achieving an optimized cross-validation accuracy of 99.5%.

By accurately predicting machine failure types such as Tool Wear Failure, Heat Dissipation Failure, Power Failure, Overstrain Failure, Random Failure, and No Failure, the system enables industries to perform proactive maintenance, reduce unexpected downtime, optimize maintenance costs, improve equipment reliability, and enhance operational efficiency.

This project demonstrates the practical application of Artificial Intelligence, Machine Learning, and IBM Cloud technologies in supporting Industry 4.0 and smart manufacturing through intelligent, data-driven predictive maintenance.
